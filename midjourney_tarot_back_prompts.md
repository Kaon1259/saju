# 타로카드 뒷면 미드저니 프롬프트 10종

## 공통 파라미터
- `--ar 2:3` : 타로카드 비율 (세로가 긴 직사각형)
- `--s 750` : 스타일라이즈 높게 (더 화려하게)
- `--v 6.1` : 미드저니 최신 버전
- `symmetrical` + `perfectly mirrored` : 카드 뒷면 상하 대칭 필수
- `no text` : 텍스트 없이 패턴만
- `flat card illustration` : 평면 카드 느낌 유지
