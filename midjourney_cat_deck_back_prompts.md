# 고양이 덱 전용 카드 뒷면 프롬프트 4종

## 공통 파라미터
- `--ar 2:3` : 타로카드 비율
- `--s 750` : 스타일라이즈 높게
- `--v 6.1` : 미드저니 최신 버전
- `symmetrical` + `perfectly mirrored` : 상하좌우 대칭 필수
- `no text` : 텍스트 없음
- `flat card illustration` : 평면 카드 느낌

## 팔레트 기준 (cat.png 레퍼런스)
라벤더 퍼플 + 핑크 풀문 + 골드 오너먼트 + 벚꽃 + 별빛 파티클
